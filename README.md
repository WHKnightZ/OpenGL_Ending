# OpenGL Ending

Ending - Kết Thúc

## Content:

Written by C programming language, use graphic library OpenGL/GLUT

### Setup:

- Setup MinGW/GCC
- Open main.cpp
- Only build main.cpp

Linker:

- Window 32 bit: -lopengl32 -lglu32 -lglut32 (for headers gl.h, glu.h và glut.h)
- Ubuntu: -lGL -lGLU -lglut (for headers gl.h, glu.h, glut.h)
